# Implementation Guidance

## Profile Usage

### BHA Patient Profile
- Must include Client ID (10 characters max)
- May include Medicaid ID when applicable
- Requires ethnicity extension

### BHA Encounter Profile  
- Must reference BHA Patient
- Requires action type (admission, update, discharge)
- Must specify service provider organization
